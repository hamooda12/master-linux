# 📖 About

This repository documents my Linux learning journey from the fundamentals to practical Linux administration and DevOps.

The goal is not to memorize commands, but to understand how Linux works through theory, labs, troubleshooting, projects, and realistic production scenarios.

Every chapter includes:

- 📖 Lessons
- 💻 Hands-on labs
- 🧪 Challenges
- 📋 Cheat sheets
- 📝 Documentation
- 💼 Interview preparation
- 📁 Portfolio projects

# 🚑 Linux Incident Bootcamp

One of the core parts of this repository is a production-style troubleshooting bootcamp.

Each incident follows a structured workflow:

```text
Observe
   ↓
Investigate
   ↓
Verify
   ↓
Fix
   ↓
Postmortem
```

Topics include:
- Configuration drift
- Broken backups
- Archive recovery
- Permission issues
- Failed services
- Log analysis
- Process troubleshooting
