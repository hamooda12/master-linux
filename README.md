## 📖 About

This repository contains my personal Linux learning journey.

The goal is not only to learn Linux commands but also to understand **how Linux works internally** through hands-on practice, real-world scenarios, and DevOps-oriented exercises.

Everything inside this repository was written while practicing on a real Ubuntu installation.

---

## 🎯 Learning Objectives

- Learn Linux from the ground up
- Master the Linux command line
- Understand the Linux filesystem
- Work comfortably inside the terminal
- Build strong Bash skills
- Prepare for DevOps and Cloud Engineering
- Document everything professionally
