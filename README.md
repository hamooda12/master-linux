# Linux Essentials - Phase 1

This repository contains my hands-on Linux practice while learning Linux
fundamentals through real-world scenarios. The goal is to build a strong
foundation in Linux system administration before moving into shell
scripting, networking, automation, and DevOps.

------------------------------------------------------------------------

# Chapter 01 -- Linux Filesystem & File Management

## Lessons Covered

1.  Navigation (`pwd`, `cd`)
2.  Listing Files (`ls`)
3.  Creating Files & Directories (`touch`, `mkdir`)
4.  File Operations (`cp`, `mv`, `rm`, `rmdir`)
5.  Viewing File Information (`file`, `stat`)
6.  Permissions & Ownership (`chmod`, `chown`, `chgrp`)
7.  Links (Hard & Symbolic)
8.  Special Permissions (SUID, SGID, Sticky Bit)
9.  Linux Filesystem Hierarchy (FHS)
